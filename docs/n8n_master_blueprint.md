## 2. Neural Logic Mastery (Skills)

### A. Expression Syntax (`{{ }}`)
-   **Braces**: Always wrap in `{{ }}`.
-   **Root**: Webhook data is **NOT** at root. Use `{{$json.body.field}}`.
-   **Spaces**: Use brackets for spaces: `{{$node["HTTP Request"].json.data}}`.
-   **Case**: Node names are case-sensitive.

### B. JavaScript Code Node
-   **Wrap**: Always return `[{json: {}}]` format.
-   **Items**: Use `$input.all()` for batch processing.
-   **Webhook**: Access via `$input.first().json.body`.
-   **Helpers**: Use `$helpers.httpRequest()` for internal calls.

### C. Validation Loop
-   **Iterate**: 2-3 cycles are normal.
-   **Thinking**: Spend ~23s analyzing errors.
-   **Fixing**: Spend ~58s applying fixes.
-   **Profiles**: Use `runtime` for deployment, `ai-friendly` for generation.
